# Fotos del estudio

> Pendiente — el sitio actualmente usa placeholders SVG (en `nosotros/page.tsx`).

Cuando el cliente provea fotos, guardarlas acá con esta nomenclatura:

- `YYYY-MM-DD_estudio_sillon.jpg`
- `YYYY-MM-DD_estudio_autoclave.jpg`
- `YYYY-MM-DD_estudio_mostrador.jpg`
- `YYYY-MM-DD_estudio_joyeria.jpg`
- `YYYY-MM-DD_estudio_atelier.jpg`
- `YYYY-MM-DD_estudio_entrada.jpg`
- `YYYY-MM-DD_piercer_trabajando.jpg`
- `YYYY-MM-DD_joyeria_<pieza>.jpg`

Recomendaciones técnicas:
- Resolución: 1920×1080 o mayor
- Formato: JPG optimizado (< 500KB cada una) o WebP
- Iluminación: cálida pero que muestre bien el instrumental
- Evitar: fotos borrosas, personas sin consentimiento
