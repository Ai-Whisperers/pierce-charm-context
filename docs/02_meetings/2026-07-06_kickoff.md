## Decisiones técnicas acordadas

| Tema | Decisión | Razón |
|---|---|---|
| Stack | Next.js (App Router) + Tailwind v4 | Mismo stack que otros sitios AiW; mantenimiento más simple |
| Color scheme | Dark mode único | La paleta (3 HEX oscuros) implica fondo oscuro |
| Hosting | `piercecharm.paragu-ai.com` | Subdominio estándar de Ai-Whisperers |
| Tipografía display | Cinzel | Estética gótica/medieval, mayúsculas con tracking |
| Tipografía script | Tangerine | Para acentos cursivos (𓆩 tu historia 𓆪) |
| Tipografía body | Inter | Limpia y legible |
| Idioma | es-PY | El cliente habla en español paraguayo |
| Decoraciones | SVG inline, no interactivas | Mejor rendimiento + control de color por CSS variables |
| Emojis | Sí, en eyebrows y separadores | Pidió explícitamente |
