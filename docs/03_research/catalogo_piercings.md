## Materiales

- **Titanio ASTM F136** (color natural o PVD negro) — primera elección
- **Oro 14k / 18k** (sólido) — para quienes lo prefieran o tengan sensibilidad
- **Acero quirúrgico 316L** — para joyería posterior (NO usar en perforaciones nuevas)
- **Niobio** — alternativa para alergias múltiples

## Estándares APP (a respetar)

- Esterilización: autoclave clase B
- Instrumental: descartable o esterilizado entre clientes
- Aguja hueca (catéter) nunca piercing gun (especialmente en cartílago)
- Asesoramiento pre-servicio obligatorio
- Cuidado post-servicio explicado por escrito + seguimiento
