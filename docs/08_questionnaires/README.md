## Cómo usar

### Cuestionarios archivados (1-8)

Están respondidos. Las respuestas están anotadas **directamente al pie** de cada cuestionario, después de la línea `---`. Ivan/Iván los lee para tener la imagen completa. No se reenvían al cliente.

### Cuestionarios activos (9-12)

Activarlos cuando se dispara el trigger correspondiente:

- **9 (Inventario):** Activar el día que el cliente confirme compra del primer stock.
- **10 (Legal):** Cuando se acerque la apertura formal del estudio (probablemente 2-4 semanas después del lanzamiento del sitio).
- **11 (UX):** Cada vez que aparece un E1-E16 nuevo en una transcripción. Aplicar lo aprobado, archivar la sección.
- **12 (Redes):** Cuando la dirección física esté confirmada y Luana esté lista para crear cuentas.

Cada cuestionario está pensado para una sesión de 20-30 min con el cliente.

### Estructura estándar de cada cuestionario

```
# Cuestionario NN — Título

> Día N. ~N preguntas, ~N min.
> Bloque: ...

---

## Secciones por bloques temáticos.

Marcadores de respuesta: _[respuesta]_

## Próximos pasos

Cuando termine, ir a STATUS.md.
```
